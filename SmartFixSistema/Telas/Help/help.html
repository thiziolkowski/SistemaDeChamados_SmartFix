1.Introdução

Este manual tem como objetivo auxiliar administradores na utilização do sistema de gerenciamento de chamados. O sistema permite o gerenciamento de chamados, bem como o cadastramento de localizações, salas e máquinas para facilitar a organização e o atendimento.

2.Requisitos de sistema

  • Navegador atualizado (Google Chrome, Mozilla Firefox, Microsoft Edge).
  • Conexão estável com a internet.
  • Resolução de tela mínima: 1366x768.

3.Login no sistema

Antes de acessar a interface principal do sistema, é necessário efetuar o login.

Passo a passo para login:

  1. Abra o navegador de sua preferência.
  2. Insira o endereço do sistema na barra de navegação.
  3. Na tela de login exibida:

    4. Digite seu usuário no campo correspondente.

    Digite sua senha no campo logo abaixo.

  4. Clique no botão Entrar localizado abaixo dos campos de entrada.
  5. Caso as credenciais estejam corretas, você será redirecionado para a tela de dashboard.

    6. Nota: Em caso de erro no login, verifique suas credenciais ou entre em contato com o suporte técnico.

4.Interface Principal

  • Dashboard

    • Após o login, o administrador será direcionado ao painel principal, onde poderá visualizar:

    • Resumo dos chamados.
    • Chamados não atendidos.
    • Chamados abertos por mês.
    • Chamados por categoria.
    • Chamados por itens.
    • Media do atendimento dos chamados.
  • Navbar

    • Localizada no canto inferior esquerdo, a navbar apresenta as seguintes opções:

    1. Home: Redireciona para a tela de dashboard.
    2. Consultar Chamados: Exibe a lista de chamados para visualização e consulta.
    3. Atualizar Chamado: (Futura implementação)..
    4. Cadastro:
      • - Cadastro de Localização: Redireciona para a tela de cadastramento de localizações.
      • - Cadastro de Sala: Redireciona para a tela de cadastramento de salas.
      • - Cadastro de Máquina: Redireciona para a tela de cadastramento de máquinas.
    5. Modo Escuro: Ativa/desativa o modo escuro para uma interface mais confortável.

5.Funcionalidades

  • Consultar chamados

    • Acesse esta funcionalidade para visualizar uma lista de chamados cadastrados.

  • Cadastro de Localização
    1. Clique em Cadastro > Cadastro de Localização na navbar.
    2. Digite a localização desejada no campo superior.
      • - Salvar: Salva a localização inserida.
      • - Limpar: Limpa a lista exibida abaixo.
      • - Buscar: Permite localizar uma localização específica.
      • - Atualizar: Atualiza informações da localização selecionada.
    3. Para editar uma localização, clique no ID correspondente na lista exibida abaixo.
  • Cadastro de Sala
    1. Clique em Cadastro > Cadastro de Sala na navbar.
    2. Insira a localização e a sala desejadas nos campos superiores.
      • - Salvar: Salva a sala cadastrada.
      • - Limpar: Limpa a lista exibida abaixo.
      • - Buscar: Permite localizar uma sala específica.
      • - Atualizar: Atualiza informações da sala selecionada.
    3. Para editar uma sala, clique no ID correspondente na lista exibida abaixo.
  • Cadastro de Máquina
    1. Clique em Cadastro > Cadastro de Máquina na navbar.
    2. Insira a localização, a sala e a identificação da máquina nos campos superiores.
      • - Salvar: Salva a máquina cadastrada.
      • - Limpar: Limpa a lista exibida abaixo.
      • - Buscar: Permite localizar uma máquina específica.
      • - Atualizar: Atualiza informações da máquina selecionada.
    3. Para editar uma máquina, clique no ID correspondente na lista exibida abaixo.

    6.Modo Escuro

    Para ativar o modo escuro:

    1. Clique no botão Modo Escuro localizado na navbar.
    2. A interface mudará para tons mais escuros, proporcionando maior conforto visual.
